# フォームテンプレート

入力フォームを生成するためのテンプレート。

## テンプレート

```
「{フォーム名}フォームを作成。

技術要件:
- shadcn/uiのForm, Input, Select, Button等使用
- Tailwind CSS
- ダークモード対応
- 日本語ラベル・プレースホルダー
- TypeScript対応
- react-hook-form + zod でバリデーション

フィールド:
- {フィールド1}: {型} - {バリデーション}
- {フィールド2}: {型} - {バリデーション}
- {フィールド3}: {型} - {バリデーション}

機能:
- 送信ボタン（ローディング状態対応）
- エラーメッセージ表示
- {追加機能}

レイアウト:
- {カラム数}
- モバイルは1カラム」
```

## 例: ユーザー登録フォーム

```
「ユーザー登録フォームを作成。

技術要件:
- shadcn/uiのForm, Input, Select, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語ラベル・プレースホルダー
- TypeScript対応
- react-hook-form + zod でバリデーション

フィールド:
- 名前: string - 必須、2文字以上
- メールアドレス: email - 必須、メール形式
- 部署: select - 必須、選択肢（営業/開発/人事/その他）
- 役割: select - 必須、選択肢（管理者/一般）

機能:
- 送信ボタン（ローディング状態対応）
- エラーメッセージ表示（フィールド下）
- キャンセルボタン

レイアウト:
- 2カラム（名前・メール / 部署・役割）
- モバイルは1カラム」
```

## 例: 検索フィルターフォーム

```
「検索フィルターフォームを作成。

技術要件:
- shadcn/uiのInput, Select, DatePicker, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語

フィールド:
- キーワード: text - 任意
- ステータス: select - 任意、選択肢（すべて/進行中/完了/保留）
- 期間: date-range - 任意

機能:
- 検索ボタン
- リセットボタン
- 即時検索（デバウンス500ms）

レイアウト:
- 横並び（フレックス）
- モバイルは縦積み」
```
