# テーブルテンプレート

データ一覧表示テーブルを生成するためのテンプレート。

## テンプレート

```
「{データ名}一覧テーブルを作成。

技術要件:
- shadcn/uiのTable, Badge, Button, DropdownMenu使用
- Tailwind CSS
- ダークモード対応
- 日本語ヘッダー
- TypeScript対応

列構成:
- {列1}: {型/表示形式}
- {列2}: {型/表示形式}
- {列3}: {型/表示形式}
- 操作: {アクション一覧}

機能:
- {ソート/フィルター/ページネーション等}
- 行ホバーエフェクト
- {追加機能}

レスポンシブ:
- モバイルで横スクロール or カード表示」
```

## 例: ユーザー一覧

```
「ユーザー一覧テーブルを作成。

技術要件:
- shadcn/uiのTable, Badge, Button, DropdownMenu, Avatar使用
- Tailwind CSS
- ダークモード対応
- 日本語ヘッダー
- TypeScript対応

列構成:
- ユーザー: アバター + 名前 + メール（2行）
- 部署: テキスト
- 役割: Badge（管理者=blue、一般=gray）
- ステータス: Badge（アクティブ=green、非アクティブ=red）
- 最終ログイン: 相対時間（3時間前）
- 操作: ドロップダウン（編集/削除/無効化）

機能:
- ヘッダークリックでソート
- 行ホバーで背景色変化
- 行クリックで詳細へ遷移

レスポンシブ:
- モバイルで横スクロール
- 固定幅ではなくmin-width使用」
```

## 例: 注文履歴

```
「注文履歴テーブルを作成。

技術要件:
- shadcn/uiのTable, Badge, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語

列構成:
- 注文番号: リンク（#ORDER-0001形式）
- 日時: YYYY/MM/DD HH:mm
- 顧客名: テキスト
- 金額: 右寄せ、¥フォーマット
- ステータス: Badge（処理中=yellow、発送済=blue、完了=green、キャンセル=red）
- 操作: 詳細ボタン

機能:
- ページネーション（10件/ページ）
- ステータスでフィルター

レスポンシブ:
- モバイルはカード形式に切り替え」
```
