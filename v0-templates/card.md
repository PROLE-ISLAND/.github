# カードテンプレート

情報表示カードを生成するためのテンプレート。

## テンプレート

```
「{カード名}カードを作成。

技術要件:
- shadcn/uiのCard, Badge, Button等使用
- Tailwind CSS
- ダークモード対応
- 日本語
- TypeScript対応

構成:
- ヘッダー: {タイトル、アイコン、アクション等}
- コンテンツ: {メイン情報}
- フッター: {補足情報、ボタン等}

機能:
- {ホバー効果}
- {クリック動作}
- {追加機能}」
```

## 例: 統計カード

```
「ダッシュボード用統計カードを作成。

技術要件:
- shadcn/uiのCard使用
- Tailwind CSS
- ダークモード対応
- 日本語
- TypeScript対応

構成:
- ヘッダー: ラベル（小さめ、muted）+ アイコン（右上）
- コンテンツ: 数値（大きめ、bold）
- フッター: 前期比（+12%↑ or -5%↓）、色分け

Props:
- label: string
- value: number | string
- change: number (%)
- icon: LucideIcon

機能:
- 増加=green、減少=red の色分け」
```

## 例: プロフィールカード

```
「ユーザープロフィールカードを作成。

技術要件:
- shadcn/uiのCard, Avatar, Badge, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語

構成:
- ヘッダー: アバター（中央）+ 名前 + 役職
- コンテンツ: メール、電話、部署
- フッター: 編集ボタン、メッセージボタン

機能:
- アバターはイニシャルフォールバック
- オンライン状態のインジケーター（緑の点）」
```

## 例: 商品カード

```
「ECサイト用商品カードを作成。

技術要件:
- shadcn/uiのCard, Badge, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語

構成:
- 画像: アスペクト比4:3、ホバーでズーム
- バッジ: SALE, NEW等（画像上に重ねて表示）
- タイトル: 商品名（2行まで、省略）
- 価格: 通常価格 + セール価格（取り消し線）
- ボタン: カートに追加

機能:
- ホバーでシャドウ強調
- カートボタンはホバー時のみ表示（デスクトップ）」
```
