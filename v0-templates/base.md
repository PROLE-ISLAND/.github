# 基本テンプレート

すべてのv0プロンプトの基盤となるテンプレート。

## テンプレート

```
「{コンポーネント名}を作成。

技術要件:
- shadcn/uiコンポーネント使用
- Tailwind CSS
- ダークモード対応（dark:クラス使用）
- 日本語テキスト
- TypeScript対応

機能要件:
- {機能1}
- {機能2}
- {機能3}

デザイン要件:
- {レイアウト}
- {配色（CSS変数使用）}
- レスポンシブ対応（モバイル優先）」
```

## 例

```
「ユーザープロフィールカードを作成。

技術要件:
- shadcn/uiのCard, Avatar, Badgeコンポーネント使用
- Tailwind CSS
- ダークモード対応（dark:クラス使用）
- 日本語テキスト
- TypeScript対応

機能要件:
- ユーザー名、メール、役割を表示
- アバター画像（フォールバックはイニシャル）
- ステータスバッジ（アクティブ/非アクティブ）

デザイン要件:
- 横長カードレイアウト
- 左にアバター、右に情報
- レスポンシブ対応（モバイルは縦積み）」
```
