# ダッシュボードテンプレート

統計・KPI表示用ダッシュボードを生成するためのテンプレート。

## テンプレート

```
「{ダッシュボード名}ダッシュボードを作成。

技術要件:
- shadcn/uiのCard, Tabs等使用
- Tailwind CSS
- ダークモード対応
- 日本語
- TypeScript対応
- Recharts（グラフ用）

セクション構成:
1. {KPIカード群}
2. {メインチャート}
3. {サブ情報}

グリッドレイアウト:
- デスクトップ: {カラム数}
- タブレット: {カラム数}
- モバイル: 1カラム」
```

## 例: 売上ダッシュボード

```
「売上分析ダッシュボードを作成。

技術要件:
- shadcn/uiのCard, Tabs使用
- Tailwind CSS
- ダークモード対応
- 日本語
- Recharts（グラフ用）

セクション構成:
1. KPIカード（4つ横並び）:
   - 売上合計（¥）、増減%
   - 注文数、増減%
   - 平均注文額、増減%
   - 新規顧客数、増減%

2. メインチャート:
   - 折れ線グラフ（売上推移、過去12ヶ月）
   - 期間切り替えタブ（週/月/年）

3. サブ情報:
   - 売上トップ5商品（横棒グラフ）
   - 最近の注文一覧（5件）

グリッドレイアウト:
- デスクトップ: KPI 4列、チャート 2列
- モバイル: 全て1列」
```

## 例: 採用ダッシュボード

```
「採用管理ダッシュボードを作成。

技術要件:
- shadcn/uiのCard, Progress使用
- Tailwind CSS
- ダークモード対応
- 日本語
- Recharts

セクション構成:
1. KPIカード（3つ）:
   - 応募者数（今月）
   - 面接予定
   - 内定率

2. パイプラインファネル:
   - 応募 → 書類選考 → 面接 → 最終 → 内定
   - 各ステージの人数と変換率

3. サブ情報:
   - 今週の面接スケジュール
   - 注目候補者リスト

グリッドレイアウト:
- デスクトップ: 3-2-2
- モバイル: 1列」
```
