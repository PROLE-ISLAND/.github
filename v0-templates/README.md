# v0 プロンプトテンプレート集

[v0.dev](https://v0.dev) でUIコンポーネントを生成する際のテンプレート集。

## 使い方

1. 目的に合ったテンプレートを選択
2. `{...}` 部分をプロジェクト固有の内容に置換
3. v0.dev に貼り付けて生成
4. 生成されたコードをレビュー・修正

## 必須要素（全テンプレート共通）

すべてのプロンプトに以下を含めること:

```
- shadcn/uiコンポーネント使用
- Tailwind CSS
- ダークモード対応（dark:クラス使用）
- 日本語テキスト
- TypeScript対応
```

## 禁止パターン

| NG | 理由 | 代替 |
|----|------|------|
| `text-red-500` 等のハードコード色 | ダークモード非対応 | CSS変数 |
| インラインスタイル | 保守性低下 | Tailwindクラス |
| 画像URLの直接埋め込み | 本番で動かない | props経由 |
| `any` 型 | 型安全性なし | 適切な型定義 |
