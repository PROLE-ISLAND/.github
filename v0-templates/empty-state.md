# 空状態テンプレート

データがない場合の表示を生成するためのテンプレート。

## テンプレート

```
「{コンテキスト}の空状態表示を作成。

技術要件:
- shadcn/uiのCard, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語テキスト
- TypeScript対応

内容:
- アイコン: {Lucideアイコン名}
- タイトル: 「{メインメッセージ}」
- 説明: 「{補足説明}」
- CTA: 「{ボタンテキスト}」ボタン

デザイン:
- 中央寄せ
- 控えめな色使い（muted）
- 適度な余白」
```

## 例: 候補者なし

```
「候補者一覧の空状態表示を作成。

技術要件:
- shadcn/uiのCard, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語テキスト
- TypeScript対応

内容:
- アイコン: Users（Lucide）
- タイトル: 「候補者がまだいません」
- 説明: 「新しい候補者を追加して、採用プロセスを開始しましょう。」
- CTA: 「候補者を追加」ボタン（primary）

デザイン:
- 中央寄せ
- アイコンは大きめ（48px）、muted色
- カード内に配置、padding多め」
```

## 例: 検索結果なし

```
「検索結果の空状態表示を作成。

技術要件:
- shadcn/uiのCard, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語

内容:
- アイコン: Search（Lucide）
- タイトル: 「検索結果が見つかりません」
- 説明: 「別のキーワードで検索するか、フィルター条件を変更してください。」
- CTA: 「フィルターをリセット」ボタン（outline）

デザイン:
- 中央寄せ
- イラストなし、シンプル」
```

```
「データ取得エラーの状態表示を作成。

技術要件:
- shadcn/uiのCard, Button使用
- Tailwind CSS
- ダークモード対応
- 日本語

内容:
- アイコン: AlertCircle（Lucide）、destructive色

- CTA: 「再試行」ボタン（outline）

デザイン:
- 中央寄せ
- エラー色は控えめに（アイコンのみ）」
```
