# PROLE-ISLAND

GitHub中心の開発体制で運営されている組織です。

## 開発フロー

1. **Issue作成** → 優先度・DoD Level選択
2. **ブランチ作成** → `feature/issue-{番号}-{説明}`
3. **開発** → Claude Code / Copilot
4. **PR作成** → `closes #{番号}` でIssue紐付け
5. **CI/品質ゲート** → 自動チェック
6. **レビュー・マージ** → 自動デプロイ

## 品質基準 (DoD)

| レベル | カバレッジ | 用途 |
|-------|-----------|------|
| Bronze | 80%+ | プロトタイプ |
| Silver | 85%+ | 開発版（推奨） |
| Gold | 95%+ | 本番品質 |
